# 👥 WeGo-Together

</div>

WeGo-Together는 사용자들이 다양한 모임을 찾고, 참여하고, 관리할 수 있도록 돕는 **모임 기반 커뮤니티 플랫폼**입니다.  
모임 추천부터 찜 기능, 사용자 맞춤 정보 관리까지 제공하여 더 쉽고 빠르게 모임을 탐색할 수 있습니다.

<br>

## 🚀 주요 기능

### 🔐 로그인 / 회원가입

- 이메일 기반 회원가입 / 로그인
- 사용자 인증 및 보안 처리

### 📋 모임 목록 페이지

- 전체 모임 목록 조회
- 카테고리/필터 기반 탐색 지원

### 📌 모임 상세 페이지

- 모임 상세 정보 확인
- 모임 참가/취소 기능
- 호스트 정보 및 구성원 확인

### ❤️ 찜한 모임 페이지

- 관심 있는 모임을 저장하고 관리
- 찜한 모임 목록 조회

### 👤 마이 페이지

- 내 정보 조회 및 수정
- 참여한 모임 내역 확인
- 사용자 설정 관리

### ⭐ 리뷰 페이지

- 모임 활동 후 리뷰 작성 및 조회
- 리뷰 평점 제공
